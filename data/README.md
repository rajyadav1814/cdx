# CDX — CSIE Sample Data

**Disclaimer:** All data generated for POC demonstration purposes.
Artist names are real public figures from Kworb Latin charts.
All financial, streaming, and social metrics are **simulated**
and do not represent actual commercial relationships or endorsements.

---

## Files

### artists.csv
100 Latin music artists sourced from Kworb Spotify charts.

| Column | Description |
|---|---|
| artist_id | Unique ID (ART_001 – ART_100) |
| name | Artist name |
| country | Origin country (ISO code) |
| genre | Primary genre |
| label | Record label |
| spotify_monthly_listeners | Monthly listeners (simulated) |
| social_blade_followers | Instagram followers (simulated) |
| youtube_subscribers | YouTube subscribers (simulated) |

**Produced by:** `generate_data.py`
**Consumed by:** Scoring engine, all 4 agents

---

### spotify_charts.csv
300 rows — chart positions across 8 territories over 5 weeks.

| Column | Description |
|---|---|
| date | Week date (YYYY-MM-DD) |
| territory | MX, CO, AR, BR, ES, CL, PE, US-Latin |
| artist_id | Foreign key → artists.csv |
| artist_name | Denormalized name |
| track_title | Track name |
| chart_position | Chart rank |
| streams_estimate | Daily streams estimate |
| peak_position | Historical peak position |
| weeks_on_chart | Weeks charted |

**Produced by:** `generate_data.py` (uses live Kworb data when available)
**Consumed by:** `scoring_engine.py`, Agent 1

---
